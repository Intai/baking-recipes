# Baking Recipes

A Claude Code plugin for following baking recipes step by step.

## Usage

Use the [`/start-recipe`](plugins/baking-recipes/commands/start-recipe.md) command followed by a recipe name to walk through it one step at a time. A timer is set automatically when a step requires waiting. e.g.
```
/start-recipe cheese scone
```
